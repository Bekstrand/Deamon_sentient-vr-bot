# OCI Free Tier Distributed Architecture

## Project Overview
Deamon_sentient-vr-bot is a distributed VR bot system deployed on Oracle Cloud Infrastructure (OCI) free tier.

## Architecture
- **Backend**: OCI Compute Instance (Always Free eligible) running Python with pandas
- **Data Processing**: Pandas for analytics and data manipulation
- **Clients**: Android/Termux devices connecting via REST APIs
- **Database**: OCI Always-Free PostgreSQL or MySQL
- **Container Runtime**: Docker on OCI compute instance
